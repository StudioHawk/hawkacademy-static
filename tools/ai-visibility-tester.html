New to API keys? Read this first (2 min)

What's an API key?

Think of it as a personal access code for an AI's "engine." When you chat on chatgpt.com, you're using the website. An API key lets this tool talk directly to ChatGPT's (or Claude's, or Gemini's) engine using your own account — so you see real results for YOUR business, not made-up ones.

Is it expensive?

No. A full test (24 queries across all 3 AIs) costs roughly $0.02–$0.05 total, charged directly to your AI account. Google Gemini is actually free for most usage. Most providers also give you free credits when you sign up.

Is it safe?

Yes. Your keys live only in your browser — they're never sent to us. You can delete or rotate them anytime in your AI account dashboard. You can also set a spending cap on each account for extra peace of mind.

How to get each key (5-10 minutes each)

Anthropic API key (for Claude)
  1. Go to console.anthropic.com and sign up (this is different from Claude.ai — but you can use the same email).
  2. Click Settings → Billing and add $5 credit.
  3. Click Settings → API Keys → Create Key. Copy the key (starts with sk-ant-) and paste it below.
OpenAI API key (for ChatGPT)
  1. Go to platform.openai.com and sign in with your ChatGPT account.
  2. Click Settings → Billing and add a payment method (minimum $5 credit).
  3. Click API keys → Create new secret key. Copy (starts with sk-) and paste below.
Google Gemini API key (usually free)
  1. Go to aistudio.google.com/app/apikey and sign in with your Google account.
  2. Click Create API key (no payment needed — Google's free tier covers most usage).
  3. Copy the key (starts with AI) and paste below.

You only need ONE key to run the test. Start with whichever is easiest (Gemini is free). Add more later for a more complete picture.
