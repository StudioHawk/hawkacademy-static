Claude SEO Skills

18 free Claude skills and prompts that audit your site, find content gaps, and generate action plans. Download the .md file, add it to a Claude Project, and run it on your own website. Each skill works with a URL (if Claude has web access) or with pasted content.

What are Claude SEO skills?

Claude SEO skills are pre-built system prompts you can drop into a Claude Project. Each one turns Claude into a specialist for a specific SEO task — a trust auditor, a content gap finder, a title tag optimiser, an AI visibility checker — without you needing to write the prompt yourself.

They're the same frameworks we use at StudioHawk across 500+ client campaigns, packaged as .md files anyone can use. No API keys, no code, no subscription.

How to use them

  1. Pick a skill below and download the .md file.
  2. Create a new Claude Project and paste the skill into the custom instructions.
  3. Start a chat in that project and follow the skill's prompts. Most take 60 seconds to 5 minutes to run.

Every skill on this page is free, forever. If you find them useful, enrol in Hawk Academy for the full SEO course that teaches you the theory behind them.
